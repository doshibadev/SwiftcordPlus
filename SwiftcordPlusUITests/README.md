# SwiftcordPlus

A SwiftUI macOS Discord client inspired by Swiftcord.  
Fully native UI for macOS, built with Swift 5 and SwiftUI.  

## Features (Planned)
- Server sidebar with icons
- Channel list for each server
- Chat view with scrollable messages
- Message input and sending
- MVVM architecture for clean code
- Preview support via SwiftUI Canvas

## Requirements
- macOS 13 or later
- Xcode 15+
- Swift 5.9+
